# conventional-fizzbuzz

이 프로젝트는 fizzbuzz와 git의 conventional commit을 연습하는 저장소입니다.

[See Demo](https://www.google.com/)

## Prerequisites

-python==3.11.0
-fastapi

## How to start
```shell
$ git clone {username/repo addr}
$ cd {repo addr}
```

$ python fizzzbuzz.py

## Installation

```shell
$ poetry install

or

$ pip install -r requirments.txt
```

## Features

- Print fizzbuzz
- Print fizz
- Print buzz
